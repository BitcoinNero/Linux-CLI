# Linux-Cli

## Download Linux-Cli and Blockchain Import
Login with root and download on root directory
```
wget https://download.btcnero.org/nero-linux.sh
chmod nero-linux.sh
./nero-linux.sh
```
Client will be install at /root/nero
## Run Daemon
```
./bitcoinnerod
```

## Create wallet
```
./simplewallet
```
